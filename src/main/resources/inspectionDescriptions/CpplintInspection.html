Runs cpplint against C/C++ files to check code style compliance with Google's C++ Style Guide.

This inspection detects violations such as:

  • Missing or incorrect header guards
  • Whitespace issues (indentation, trailing spaces, missing newlines)
  • Include order violations
  • Line length violations
  • Naming convention issues
  • And many more style violations

Configure the cpplint executable path and options in Settings | Cpplint.

Some violations can be automatically fixed using quick fixes.
